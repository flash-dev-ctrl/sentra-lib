# Changelog

All notable changes to this project will be documented in this file.

## 0.1.2

- Added GitHub release readiness for the Rust library.
- Kept C binding boundary tests compatible with Windows checkout line endings.

## 0.1.0

- Added the initial reusable Sentra Rust library for local AI Agent discovery.
- Added asset readers for skills, providers, MCP servers, memory, and cron entries.
- Added YARA, hash, local threat-intelligence, optional online threat-intelligence, and optional LLM risk checks.
- Added optional C ABI support behind the `c-binding` feature.
