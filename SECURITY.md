# Security Policy

`sentra-lib` reads local AI Agent assets and powers Sentra risk scanning, so
security reports are especially welcome.

## Supported Versions

Security fixes are applied to the latest release line.

## Reporting A Vulnerability

Please do not disclose vulnerabilities publicly before maintainers have had
time to investigate.

Include:

- Affected version or commit.
- Operating system and integration method.
- Reproduction steps.
- Impact and any known workaround.

If the project is mirrored to GitHub, use GitHub private vulnerability
reporting when available. Otherwise, contact the maintainers through the
repository owner.

## Scope

Reports are in scope when they affect Agent discovery, asset parsing, rule
loading, risk checks, C ABI boundaries, or serialized scan output.
